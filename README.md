## Features

- **Dynamic Product Display** - Products fetched from Fake Store API with real-time loading
- **Product Categories** - Filtered sections for Electronics and Jewelries
- **Shopping Cart** - Add and remove items with instant updates
- **Responsive Design** - Optimized for mobile, tablet, and desktop views
- **Interactive Cards** - Hover effects with smooth transitions and product details
- **LocalStorage Support** - Cart data persists across browser sessions
- **Product Ratings** - Display star ratings and review counts
- **Image Fallback** - Placeholder images for missing product images

## How to Run Locally

1. **Clone this repository:**
```bash
git clone https://github.com/sereitepy/aditi-week-2.git
```

2. **Navigate to the project folder:**
```bash
cd aditi-week-2
```

3. **Open in VS Code or any code editor:**
```bash
code .
```

4. **Install the Live Server extension:**
   - Search for "Live Server" by Ritwick Dey in VS Code extensions
   - Install it

5. **Run the project:**
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - The site will open in your browser at `http://localhost:5500`
